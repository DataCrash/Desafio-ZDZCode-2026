# Análise da Vaga ZDZCode

## Fonte

- [Vaga ZDZCode](https://odoo.zdzcode.com.br/pt/jobs/desenvolvedor-a-full-stack-100-remoto-8)

## Resumo Executivo

- Cargo: Desenvolvedor(a) Full Stack
- Modalidade: 100% remoto
- Regime: PJ
- Carga: 160-180h/mês
- Horário: comercial
- Contexto: sistemas de gestão hospitalar em produção

## Stack Esperada

- Back-end: C#, .NET, Entity Framework, SQL Server
- Front-end: Vue.js 3, Nuxt.js, Vuetify, JavaScript, TypeScript
- Infra/processo: Azure DevOps, Git

## Responsabilidades

- Desenvolver e manter APIs em C#/.NET
- Criar e evoluir telas com Vue.js 3, Nuxt.js e Vuetify
- Modelar dados, migrations e queries SQL
- Diagnosticar problemas e entregar soluções funcionais
- Trabalhar com Azure DevOps
- Participar de reuniões com time e clientes

## Requisitos Obrigatórios

- 2+ anos com C# e .NET em produção
- SQL e bancos relacionais
- Entity Framework
- Git e controle de versão
- Equipamento para trabalho remoto

## Requisitos Desejáveis

- Vue.js / Nuxt.js / Vuetify
- Azure ou cloud em geral
- Azure DevOps ou similares
- APIs RESTful

## Sinais Implícitos de Avaliação

- Experiência real de produção
- Autonomia com responsabilidade
- Capacidade full stack ponta a ponta
- Modelagem de dados sólida
- Pragmatismo e velocidade de entrega
- Qualidade de repositório, commits e documentação

## Observações

- A empresa enfatiza código real, clientes reais e problemas reais.
- O discurso privilegia execução pragmática sobre excesso de teoria.
- Onboarding com primeiro commit na primeira semana sugere alta importância em autonomia inicial.
